# mario
An Adaptive Mesh Refinement I/O Analyzer

## Installation
```
$ git clone git@github.com:jungwonkim/mario.git
$ cd mario
$ mkdir build
$ cd build
$ cmake .. -DCMAKE_INSTALL_PREFIX=<installation_dir>
$ make install
```
